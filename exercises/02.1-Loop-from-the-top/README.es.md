# `02.1` Loop from the top

Este bucle está `iterando` la lista de principio a fin... incrementando uno a uno:
```py
my_list = [2, 56, 9]

for item in my_list:
    print(item) # 2 -> 56 -> 9
```

## 📝 Instrucciones:

1. Vamos a intentar iterar desde el fin hasta el principio.
